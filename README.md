# Pocketbase

> Open Source backend
for your next SaaS

## Requirements

- Docker (https://docs.docker.com/get-docker/)



## Build Setup locally 

``` bash
# build
docker build -t pocketbase .

# run
docker run -it --rm -p 8080:8080
```

Open url UI in:

http://0.0.0.0:8080/_/


For more: https://www.igorprvieira.com/blog/2023-04-02-pocketbase/
